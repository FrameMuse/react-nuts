## Introduction
### Create new Popup Element

All need for creating such element is a valid JSX.Element:

```js
// Arrow function
const PopupElement = () => ":3"
// Standart function
function PopupElement() {
  return ":3"
}
```
### Using Popup Context
```js
const PopupElement = () => {
  const { Resolve, Params } = usePopupContext()
  return ":3"
}
```

### Component usage
```tsx
<PopupProvider>
    
</PopupProvider>
```
